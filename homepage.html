Introdução ao HTML e CSS

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas web. CSS (Cascading Style Sheets) é usado para estilizar essas páginas.

Estrutura do HTML

Um documento HTML começa com a declaração do tipo de documento <!DOCTYPE html>, seguida pela tag <html> que engloba todo o conteúdo da página.
